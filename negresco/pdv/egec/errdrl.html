ERRO DE REDE LOCAL


Problema de comunicação dos caixas com o concentrador / servidor/ etc.
1º Verifique o erro de rede local que ocorre no log do pdv
Caso ocora erro de Módulo PDV clique aqui
Caso ocora erro de Módulo CONC PDV clique aqui
Caso ocora erro de Módulo SGR clique aqui

Módulo pdv

Problema de comunicação do pdv, efetuar testes de comunicação no concentrador, servidor e no 10.228.0.50.
Caso comunique normalmente efetue restart do pdv.

Módulo CONC PDV

Problema de comunicação concentrador PDV, efetuar teste no concentrador de PDV.
Verifique se PDV comunica dentro do concentrador.
Caso pdv não responda ao ping dentro do concentrador:
Caso acuse Erro de Rede Local Conc PDV - Destination Host Unreachable.
Deve-se acionar o departamento de Redes Telecom e solicitar um clear na tabela ARP e Port Security dos Switchs. Incidente de exemplo onde foi aplicada solução é IN2036571
Caso pdv responda ao ping dentro do concentrador:
Eftue resart do concentrador de pdv

Módulo SGR

Acessar servidor principal da filial.
executar comandos ps -ef | grep conc e ps -ef | grep surica para verificar se o concloja e suricato estão ativos
efetuar comando sudo conclojacb.sh
Selecione a opção dois do menu (carrega serviço)
Carrege o menu do concentrador e execute as opções 2, 3, 5 e 6
